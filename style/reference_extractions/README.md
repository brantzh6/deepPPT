# Reference Extractions

Store one structured extraction file per reference slide.

Recommended naming:
- `<deck_id>--slide-<number>--<page_type_candidate>.json`

Use:
- [reference_slide_extraction.schema.json](D:/code/enterprise-ppt-system-bootstrap/schemas/reference_slide_extraction.schema.json)

Workflow:
1. capture the source slide path or screenshot path
2. describe narrative role and claim
3. extract visual anchor and alignment logic
4. record why the slide works
5. record what should not be copied blindly

Starter files:
- `template.reference-slide.json`

This folder is for structured learning artifacts, not raw screenshots.
